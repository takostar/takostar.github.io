    1. 在C:\Users\<你的用户名>\Postman\files下创建postman.txt，写入 test postman

    2. 在表单中选择 postman 文件目录下的文件，参数设为 file

    常用命令

    • ipfs add 添加文件

    • ipfs cat 打印文件

    • ipfs config show 显示设置

    • ipfs pin ls 显示本地持久化文档(通过 add 添加的默认为本地持久化的)
